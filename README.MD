# Project Description
This is the back-end part of my personal portfolio. Here I show my skills as back-end developer as well.
In this project I tried to show how I use my main stack and some key concepts pretty important for good and scalable web apps.

## Technologies
- NodeJs
- Typescript
- ExpressJs 
- Prisma
- Sqlite

## Concepts
- Clean Architecture
- Clean Code
- Scalability

# Getting Start
Install dependencies:
`npm install`

create migrations:
`npm run migrate`

run migrations:
`npm run generate`

run app:
`npm run dev`
